# Computacao_Paralela
Desenvolver em C/C++ um programa que responde qual a porcentagem de acertos e erros de uma cache de mapeamento direto que suporte 8 linhas de tamanho 1.  A entrada de dados será feita por um arquivo de texto contendo 1 endereço por linha.  Assuma que os endereços são valores inteiros entre 0 e 255.
